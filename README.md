# NoteForge

Minimal Notion-lite clone built with Phalcon and MarkForge.

## Requirements

- PHP 8.5+
- Composer

## Setup

1. Install dependencies:

   composer install

2. Create env file:

   cp .env.example .env

3. Run local server:

   php -S 127.0.0.1:8000 -t public
